# Поля выгрузки (колонки xlsx), обязательные для заполнения переменных из property.txt
# Формат выгрузки — как в test.txt (одна строка = одно дело, ключи — названия колонок)

# === Для search_case() ===
number_case
    → "ID дела  (ID/ номер убытка)" или "Наименование должника"
    (в 1С ищется по ответчику — логичнее "Наименование должника")

# === Для handle_court() — вкладка СУД ===
name_defedant
    → "Наименование должника"

court
    → "Суд"

date_base
    → "Дата вынесения решения" (дата 1 в форме)

date_plus_mounth
    → "Дата направления иска" или другая дата (дата 2 в форме; при необходимости +1 месяц к date_base)

result_case
    → "Результат судебного дела"
    маппинг: "Удовлетворены" / "Полностью удовлетворён" → удп; "Частично" → удч; "Отказ" / "Отказ в иске" → отк

summ_requests_s
    → "Сумма долга" (запрошенная сумма по основному долгу; число без пробелов)

summ_real_s
    → "Взысканная сумма (основной долг)" (одобренная сумма; число, запятая как десятичный разделитель → убрать пробелы)

summ_requests_g
    → "Сумма госпошлины" (запрошенная госпошлина; число без пробелов)

summ_real_g
    → "Взысканная сумма (расходы)" или "Сумма госпошлины" (одобренная госпошлина; число)

# === Для ip_handler() — вкладка ИП ===
view_ip_list
    → в выгрузке нет прямого поля; фиксировать по типу документа или оставить по умолчанию: "Оригинал исполнительного листа"

number_ip_list
    → "Номер исполнительного листа"

summ
    → "Взысканная сумма (основной долг)" + "Взысканная сумма (расходы)" (итого к взысканию) или "Остаток задолженности основного долга" — в зависимости от логики

data_get_ip_list
    → "Дата получения исполнительного листа"

#   Взысканная сумма (расходы)

#   Номер исполнительного листа
